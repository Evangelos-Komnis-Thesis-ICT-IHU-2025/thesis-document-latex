\section{Υποστήριξη μέσω του \ENG{PHP SDK}}
\label{sec:appendix_sdk_contract_support}

\texttt{\ENG{LaunchesApi}} και \texttt{\ENG{AttemptsApi}}, καθώς και σε \ENG{DTO classes}, \ENG{query objects},
\ENG{forms} και μηχανισμούς μεταφοράς \ENG{HTTP} αιτημάτων. Προσφέρεται ως ενδεικτικό παράδειγμα της υποστήριξης ανάμεσα στην επικοινωνία
